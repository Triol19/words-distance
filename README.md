# words-distance

Short example of algorithm for counting number of words between 2 provided words in text

## Requirements

```
Python 3.6+
```

## Usage

### Basic
```
python distance.py
```

### Params
`-f` - name of the file (`file_to_process.txt` by default)

`-ci` - case insensitive mode (`False` by default)

`-w1` - First word

`-w2` - Second word

## Running the tests

```
python -m unittest
```
